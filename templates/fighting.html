Fighting Game Ideas

    My first fighting game idea

I have always wanted to create a fighting game where parrying and grabbing where the main mechanics.

I think it would be interesting if I could create a game in which it is better to create combos out of parries and grabs/throws.

Timing parries and grabs correctly is an important part of the skillset in a lot of fighting games and as such

I figured a game which leans more heavily into these mechanics could offer a more challenging, yet rewarding experience for more experienced players.


Of course a lot of fighting games have an underlying story between characters and I must admit I thought significantly less about what sort of narrative

I would construct for any potential game I would make as in the genre the story has always been separate to the mechanics.

What I have thought a lot about is character design. I don't have any designs in particular, however, I am aware that a large part of making an enticing fighting game

is the character designs. Creating distinct figures and yet still maintaining a visually coherent style to link them together is a necessity when making any game and

fighting games are no exception.
