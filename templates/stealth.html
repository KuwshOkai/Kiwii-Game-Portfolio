Stealth Game Ideas

    General Concpets

Much like with fighting games I don't have one specific idea per se, but some concepts that i'd like to incorporate into a stealth game.

I think it would be interesting to have a stealth game in which the enemy ai learns the player's patterns in distracting them and adapt as the game progresses.

For example, if the player uses thrown items a lot as a distraction, the game could start punishing them for it by having enemies become wiser.

This would give ample incentive for player's to diversify their techniques and allow them to employ new strategies which can attribute to the replayability of the game.
